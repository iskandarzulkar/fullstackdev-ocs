# 🚀 Teknologi
## Backend: Node.js v24.18.0 & NPM v11.16.0

# PERSYARATAN

Pastikan perangkat sudah memiliki:

- Node.js dan npm
- Microsoft SQL Server
- Database SQL Server yang sudah dibuat
- User SQL Server yang memiliki akses untuk membuat tabel, stored procedure, insert, update, delete, dan select

# 🛠️ Instalasi

## 1. Clone Repository
``` bash
 git clone https://github.com/iskandarzulkar/ocs.git
 cd ocs-service
```

## 2. Instalasi 
```bash
    npm run install
```

## 3. migration database
``` bash
    npm run migrate
```

## 4. seeder database
```bash
    npm run seeder
```

## 5. running aplikasi
```bash
    npm run dev || npm start
``` 
Backend akan berjalan di:

```text
http://localhost:3131
```
